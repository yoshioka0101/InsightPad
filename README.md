# InsightPad

## 説明
- 勉強するにあたって、メモをして内容を整理するためのリポジトリが欲しかったので作成した
- 内容は[backend loadmap](https://roadmap.sh/backend)を参考にファイル構造を作成した

## 方針
- 勉強の内容は本でもいい
- 記事を上げるだけではダメ
- できるだけ毎日commitする
